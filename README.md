# Overview

MIEAI Nexus is the central gateway for a collection of experimental data systems developed under the **PAMI Core Intelligence architecture**.

The platform provides lightweight, browser-based tools that allow users to perform analytics, statistical computing, data transformation, and utility operations directly through web applications.

All systems are designed to be fast, accessible, and freely usable.

The project explores a modular approach to building data intelligence platforms using cloud-based technologies.
